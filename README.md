# Drinky - 절주 지원 앱

> "더 오래, 더 함께" - 원터치 체크로 간편하게 절주 습관을 만들어보세요.

## 소개

**Drinky**는 술을 완전히 끊기보다 건강하게 조절하고 싶은 사람들을 위한 간편한 절주 지원 앱입니다.

기존 금주 앱들의 높은 진입장벽(일기 작성 등)을 없애고, O/X 버튼 하나로 1초 만에 음주 여부를 기록할 수 있습니다.

## 주요 기능

- **원터치 체크인** - O/X 버튼으로 오늘 음주 여부 간편 기록
- **연속 절주 스트릭** - 연속 금주 일수를 시각적으로 표시하여 동기부여
- **격려 메시지** - 체크인할 때마다 따뜻한 격려 메시지 제공
- **소셜 로그인** - Google / Kakao 계정으로 간편 로그인

## 대상 사용자

- 음주량을 줄이고 싶은 20~40대 성인
- 복잡한 기록 없이 간단하게 절주 습관을 만들고 싶은 사람
- 시각적 피드백으로 동기부여를 받고 싶은 사람
