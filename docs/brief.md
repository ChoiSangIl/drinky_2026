## Executive Summary

**Drinky**는 술을 완전히 끊기보다 건강하게 조절하고 싶은 사람들을 위한 **간편한 절주 지원 애플리케이션**입니다.

- **핵심 문제:** 기존 금주 앱들은 일기 작성 등 진입장벽이 높아 지속 사용이 어려움
- **타겟 시장:** 음주량 조절을 원하는 20-40대 성인
- **핵심 가치:** 원터치 체크로 간편하게 기록하고, 시각적 피드백과 격려로 절주 습관 형성 지원
- **슬로건 방향:** "더 오래, 더 함께"

---

## Problem Statement

### 현재 상황과 페인 포인트

1. **기존 금주 앱의 한계**
   - 대부분 일기 기능 위주로 매일 작성해야 하는 부담
   - 복잡한 UI로 인한 높은 진입장벽
   - 오래된 앱들은 UI/UX가 현대적이지 않음

2. **사용자의 어려움**
   - 바쁜 일상에서 앱을 열고 기록할 시간이 없음
   - 술을 마신 날 기록하면 좌절감을 느끼게 됨
   - 혼자서 절주를 지속하기 어려움

3. **문제의 영향**
   - 음주 습관 개선 실패 → 건강 악화
   - 가족, 친구와 보내는 질 좋은 시간 감소
   - 장기적으로 삶의 질 저하

### 왜 지금 해결해야 하는가?

- 건강에 대한 관심 증가 (특히 40대 이상)
- MZ세대의 "소버 큐리어스" 트렌드
- 기존 앱들이 사용자 니즈를 충족시키지 못하고 있음

---

## Proposed Solution

### 핵심 컨셉

**"원터치로 기록하고, 함께 응원받는 절주 앱"**

기존 금주 앱들이 "일기"에 집중했다면, Drinky는 **최소한의 마찰로 기록**하고 **긍정적 피드백**을 제공하는 데 집중합니다.

### 차별화 포인트

| 기존 금주 앱 | Drinky |
|-------------|--------|
| 매일 일기 작성 필요 | 원터치 O/X 체크 |
| 무거운 톤앤매너 | 가볍고 격려하는 톤 |
| 혼자서 기록 | 선택적 친구 공유 |
| 앱을 열어야 기록 | 위젯으로 바로 체크 |

### 성공 가능성

- **간편함:** 1초 체크 → 습관 형성 용이
- **감정 설계:** 자신감 부여, 좌절감 최소화
- **사회적 요소:** 친구와 가볍게 공유하며 동기부여
- **확장 가능성:** 건강 습관 플랫폼으로 성장 가능

## Target Users

### Primary User Segment: 건강 경고를 받은 40대

**프로필:**
- 40대 직장인
- 건강검진에서 간 수치 경고를 받음
- 가족이 있으며, 건강하게 오래 살고 싶음

**현재 행동:**
- 스트레스로 매일 퇴근 후 음주
- 줄여야 한다는 건 알지만 의지만으로 어려움
- 금주 앱을 찾아보지만 일기 쓰기 부담으로 포기

**니즈와 페인 포인트:**
- 간편하게 기록하고 싶음
- "건강해질 때까지"라는 명확한 목표 필요
- 가족에게 좋은 모습 보여주고 싶음

**목표:**
- 주 5회 음주 → 주 2회로 감소
- 건강검진 수치 정상화
- 가족과 더 많은 시간 보내기

### Secondary User Segment: 회식이 많은 20-30대 직장인

**프로필:**
- 20-30대 직장인 (특히 신입~대리)
- 회식 문화가 있는 회사 근무
- 건강과 다이어트에 관심 있음

**현재 행동:**
- 회식으로 인한 불가피한 음주
- 주말에도 친구 모임으로 음주
- SNS에서 건강 관련 콘텐츠 소비

**니즈와 페인 포인트:**
- 사회생활과 건강 사이 균형
- 가볍게 공유하고 싶지만, 때로는 숨기고 싶음
- 복잡한 앱은 귀찮음

**목표:**
- 주말 과음 줄이기
- 다이어트와 병행
- 건강한 라이프스타일 유지

## MVP Scope

### Core Features (Must Have)

- **원터치 음주 체크:** 오늘 술 마셨는지 O/X로 간편하게 기록 (핵심 차별화 기능)
- **연속 금주 스트릭:** 연속으로 술 안 마신 날 카운트, 동기부여 핵심 요소
- **음주 기록 차트:** 주간/월간 음주 패턴 시각화, 인사이트 제공
- **격려 푸시 알림:** "잘하고 있어요!" 등 긍정적 메시지로 이탈 방지

### Out of Scope for MVP

- 위젯 기능 (네이티브 앱 전환 후)
- 친구 공유 및 소셜 기능
- 뱃지/업적 시스템
- 커뮤니티 기능
- Apple Health/Google Fit 연동
- 다국어 지원

### MVP Success Criteria

MVP가 성공했다고 판단하는 기준:

1. **사용성:** 원터치 체크 완료율 90% 이상 (체크 시작 → 완료)
2. **리텐션:** 7일 리텐션 25% 이상
3. **만족도:** 사용자 설문 NPS 30 이상
4. **스트릭:** 최소 100명이 7일 스트릭 달성
5. **피드백:** 치명적 UX 이슈 없이 2주 운영

## Post-MVP Vision

### Phase 2 Features

   - 홈화면에서 앱 열지 않고 바로 체크
   - iOS/Android 네이티브 앱 전환 필요

2. **친구 공유 기능**
   - 음주 기록을 친구에게 선택적 공유

   - 딥링크로 앱 바이럴 효과

3. **뱃지 시스템**
   - 7일, 30일, 100일 등 성취 뱃지
   - 게이미피케이션으로 동기부여 강화

### Long-term Vision (1-2년)

**"건강한 습관을 만드는 플랫폼"**

- 절주 → 다이어트, 운동, 수면 등 건강 습관 전체로 확장
- 종합 건강 관리 앱으로 포지셔닝
- 데이터 기반 개인화된 건강 인사이트 제공

### Expansion Opportunities

- **B2B 기업 복지:** 직원 건강 관리 솔루션으로 제공
- **보험사 연계:** 건강 데이터 기반 보험료 할인 프로그램
- **병원/클리닉 연계:** 알코올 사용 장애 관리 도구
- **글로벌 진출:** 영어권 시장 (미국의 소버 큐리어스 트렌드)

  - 네이티브 앱 개발 역량 없음 → PWA 우선
  - 위젯 기능은 MVP에서 제외

- 사용자는 간편한 기록 방식을 원한다
- 푸시 알림이 리텐션에 긍정적 영향을 미친다
- 건강 경고를 받은 40대가 가장 적극적인 사용자가 될 것이다
- PWA로도 충분한 사용자 경험을 제공할 수 있다
- 한국 시장에서 먼저 검증 후 확장한다
